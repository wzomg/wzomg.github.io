欧拉函数

发表于 2019-01-21 更新于 2019-07-18 分类于 算法笔记
本文字数： 2.5k 阅读时长 ≈ 2 分钟

定义：

对于正整数n，欧拉函数是小于n的正整数中与n互质的数的个数$(\varphi(1)=1)$。此函数以其首名研究者欧拉命名，它又称为$\varphi$函数、欧拉商数等。举个栗子：$\varphi(8)=4$，因为1,3,5,7均和8互质。
